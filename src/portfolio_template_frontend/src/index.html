About ICP

Internet Computer (ICP)
A blockchain-based protocol that aims to create a decentralized alternative to centralized internet cloud providers:

  • How it works: ICP is a network of independent data centers that can work together to create a decentralized internet. It uses smart contracts to replace most of the world's software. ICP's smart contracts are designed to be as powerful as traditional software.
  • Features: ICP can interface with other smart contract blockchains and traditional Web 2 internet resources. It also allows developers to write contracts using popular languages like Rust, TypeScript, or Python.
  • ICP token: The ICP token is used for governance, to reward network participants, and to pay fees for making transactions.
